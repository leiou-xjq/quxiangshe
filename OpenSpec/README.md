# OpenSpec 规范仓库

> 全栈开发技术规范中心 - 数据库 | 后端 | 前端

## 目录结构

```
OpenSpec/
├── proposals/    # 技术提案
├── specs/       # 技术规范
├── tasks/       # 任务文档
├── designs/     # 设计文档
└── changes/     # 变更记录
```

## 快速导航

- [ proposals/ ] - 各类技术提案和RFC
- [ specs/ ] - 完整的 技术规范文档
- [ tasks/ ] - 任务规划和追踪
- [ designs/ ] - 系统架构和UI设计
- [ changes/ ] - 版本变更日志

## 贡献指南

请参考各子目录下的 CONTRIBUTING.md 文件了解贡献流程。
