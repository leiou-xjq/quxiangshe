## 任务描述

### 背景

项目启动阶段，需要设计核心业务表的数据库结构

### 目标

设计并输出以下表结构：
1. 用户表（sys_user）
2. 角色表（sys_role）
3. 权限表（sys_permission）
4. 用户角色关联表（sys_user_role）
5. 角色权限关联表（sys_role_permission）

## 详细说明

### 设计要求

1. 遵循数据库设计规范
2. 符合第三范式
3. 包含必要的索引
4. 支持软删除
