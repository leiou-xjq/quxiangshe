## 任务状态

| 状态 | 说明 |
|-----|------|
| 待开始 | 任务未开始 |
| 进行中 | 任务正在执行 |
| 待评审 | 任务完成待评审 |
| 已完成 | 任务已完成 |
| 已取消 | 任务已取消 |

## 任务优先级

| 优先级 | 说明 |
|-------|------|
| P0 | 紧急重要 |
| P1 | 高优先级 |
| P2 | 中优先级 |
| P3 | 低优先级 |

## 任务管理流程

1. 创建任务（填写任务模板）
2. 分配执行人
3. 更新任务进度
4. 完成后提交评审
5. 关闭任务
