## 任务描述

### 背景

系统需要提供用户管理功能

### 目标

开发用户管理服务，实现用户CRUD功能

## 详细说明

### 功能需求

1. 用户列表查询（支持分页、排序、过滤）
2. 用户详情查询
3. 用户创建
4. 用户信息更新
5. 用户删除（软删除）
6. 用户状态启用/禁用

### 技术要求

- 遵循 RESTful API 设计规范
- 实现统一的错误处理
- 添加操作日志
