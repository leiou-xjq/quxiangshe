# 登录页面设计

## 概述

本文档描述登录页面的UI设计

## 页面结构

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│                   ┌─────────────────┐                      │
│                   │                 │                      │
│                   │   [Logo]        │                      │
│                   │                 │                      │
│                   │   系统名称      │                      │
│                   │                 │                      │
│                   │  ┌───────────┐  │                      │
│                   │  │ 用户名    │  │                      │
│                   │  └───────────┘  │                      │
│                   │                │                      │
│                   │  ┌───────────┐  │                      │
│                   │  │ 密码      │  │                      │
│                   │  └───────────┘  │                      │
```

## 设计规范

### 布局

- 居中布局
- 背景：渐变色或图片
- 卡片：白色背景，圆角阴影

### 配色

| 元素 | 颜色 |
-----|------|
| 背景 | #f0f2f5 |
| 卡片 | #ffffff |
| 主按钮 | #1890ff |
| 文字 | #333333 |
| 副文字 | #666666 |

### 组件

| 组件 | 尺寸 |
-----|------|
| 输入框 | 高度 40px |
| 按钮 | 高度 40px |
| 卡片宽度 | 400px |

## 功能说明

### 输入验证

- 用户名：必填
- 密码：必填，最少6位

### 记住我

- 勾选后保持登录状态7天

### 错误提示

- 用户名或密码错误显示错误提示
- 错误信息显示在表单下方
