## 功能说明

### 搜索功能

- 用户名：模糊搜索
- 邮箱：模糊搜索
- 状态：全部/启用/禁用

### 操作功能

- 新增：打开新增弹窗
- 编辑：打开编辑弹窗
- 删除：确认后删除
- 启用/禁用：切换状态

### 弹窗设计

#### 新增/编辑用户弹窗

```
┌───────────────────────────────────┐
│  新增用户                    [×] │
├───────────────────────────────────┤
│  用户名：  [________________]     │
│  邮箱：    [________________]     │
│  手机号：  [________________]     │
│  角色：    [选择 ▼]              │
│  状态：    (●) 启用  ( ) 禁用    │
├───────────────────────────────────┤
│              [取消]  [确定]       │
└───────────────────────────────────┘
```
