# 趣享社笔记模块技术文档（开发视角）

## 一、模块概览
笔记模块负责笔记的创建、浏览、互动（点赞/收藏）、评论、以及与 ES 的搜索索引同步。核心技术栈：Java、Spring Boot、MyBatis-Plus、Elasticsearch、RabbitMQ、Redis、JWT。

## 二、核心接口与功能覆盖
- 发布、编辑、删除笔记
- 列表展示（首页、用户笔记、搜索结果）
- 详情查看（包含作者、时间、图片、标签、分类）
- 点赞、收藏、评论（增删改、点赞、回复）
- ES 搜索、热门笔记排序
- 审核、状态管理、敏感词处理（后续扩展）

## 三、实现要点
- 数据模型与索引
  - MySQL：t_note、t_note_like、t_note_collect、t_note_image、t_sensitive_check_log 等表
  - ES：t_note 索引，t_user 索引用于作者信息
- 流程设计
  - 发布：敏感词检测 -> 审核状态 -> 保存笔记与图片 -> 异步推送 ES 新文档
  - 详情：检查笔记存在性、增加浏览量、聚合图片/标签/作者信息
  - 互动：点赞/收藏具备幂等性，Redis 缓存 + DB 兜底
  - 评论：创建、列出、点赞、回复等能力
  - 搜索：Elasticsearch 的全文检索 + 高亮显示
- 权限与安全
  - 访问笔记详情等公开数据不强制鉴权，关键操作如发布/点赞/评论等需认证
  - 授权控制由 Spring Security + JWT 实现

## 四、模块耦合与关联
- Auth 模块提供鉴权能力，Note 模块的创建/互动接口需要鉴权
- Search 模块提供笔记与用户搜索能力，与 Note、User 数据源整合
- RabbitMQ 用作 ES 同步和事件驱动，解耦前后端与索引层

## 五、部署与监控要点
- Redis 限流与黑名单的健康检查/降级策略
- ES 同步的重试策略与幂等性保障
- 全局异常处理与统一日志输出
